# DSA AI Platform

A MERN stack web application to help users master Data Structures and Algorithms with AI.

## Features

- AI generated DSA roadmap
- DSA topics checklist
- AI doubt solver
- Full authentication system

## Tech Stack

Frontend
- React
- Tailwind CSS

Backend
- Node.js
- Express.js

Database
- MongoDB
